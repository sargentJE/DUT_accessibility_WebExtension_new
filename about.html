About This Extension

This extension collects structured feedback about website accessibility experiences, helping developers understand real-world challenges and make the web more inclusive for everyone, including those with visual impairments, motor disabilities, and other accessibility needs.

Key Features

  • Multi-page structured feedback form
  • User experience documentation
  • Accessibility preference collection
  • Secure data submission
  • JSON data export functionality
