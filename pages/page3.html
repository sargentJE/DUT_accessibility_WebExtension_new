Website Navigation

Complete each task below and answer the related questions before moving on.

Note: Fields marked with * are required.

Task 1: Find Main Features or Services

What to do: Look for navigation menus, search functionality, or prominent links to key sections of the website (e.g., products, services).

Tip: Try using both mouse and keyboard (Tab key) navigation.

Task 2: Locate Contact Information

What to do: Find phone numbers, email addresses, contact forms, or office locations.

Tip: Check the header, footer, and "Contact" pages.

Task 3: Find Support Resources

What to do: Identify where FAQs, help documentation, or customer support options are located.

Tip: Look for "Help," "Support," or "FAQs" in the main navigation or footer.

Task 4: Return to Homepage

What to do: Test different ways to get back to the homepage (e.g., logo, home link, breadcrumbs).

Tip: Try both the website logo and explicit "Home" links.
