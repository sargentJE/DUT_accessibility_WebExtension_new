Interactive Features

Complete each task below and answer the related questions.

Note: Fields marked with * are required.

Task 1: Complete a Form

What to do: Find and attempt to complete a form (contact, signup, search).

  • Evaluate ease of locating and understanding the form.
  • Test labeling and required fields.
  • Try submitting with test data.

Tip: Use only keyboard (Tab, Enter) to navigate.

Task 2: Test Customization Options

What to do: Test options like font size, color themes, layout preferences, or language selection.

Tip: Look in headers, footers, profile settings, or accessibility menus.

Task 3: Test Error Handling

What to do: Cause errors (incorrect form entries, incomplete submission, invalid search) to test responses.

Tip: Evaluate if error messages explain what went wrong, why it matters, and how to fix it.

Task 4: Assess Feedback Clarity

What to do: Complete actions (form submission, add to cart, save preference) and evaluate feedback.

Tip: Feedback should be immediate, visible, and understandable without relying solely on color.
